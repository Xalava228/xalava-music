# 3 ВЕРСТКА И СЕРВЕРНАЯ ЧАСТЬ

## 3.1 Описание общей структуры

Веб-приложение XALAVA MUSIC построено по архитектуре клиент-сервер с разделением на три основных компонента: клиентская часть (frontend), серверная часть (backend) и база данных.

### Архитектура приложения

Приложение использует классическую трехзвенную архитектуру:

1. **Презентационный слой (Frontend)**:
   - HTML5 для структуры страниц
   - CSS3 для стилизации с использованием современных техник (CSS Variables, Flexbox, Grid)
   - Vanilla JavaScript для интерактивности и взаимодействия с API
   - Адаптивный дизайн с поддержкой мобильных устройств

2. **Бизнес-логика (Backend)**:
   - Flask (Python) как веб-фреймворк
   - Интеграция с Yandex Music API через библиотеку `yandex-music`
   - RESTful API для взаимодействия с клиентом
   - Управление сессиями пользователей

3. **Слой данных (Database)**:
   - SQLite для хранения пользовательских данных
   - Кэширование данных треков для оптимизации
   - Хранение истории поиска, прослушиваний, избранного

### Структура файлов проекта

```

├── app.py                 # Основной файл Flask-приложения
├── database.py           # Модуль работы с базой данных
```

### Принципы проектирования

1. **Single Page Application (SPA)**: Приложение использует подход SPA, где все взаимодействие происходит без перезагрузки страницы через динамическое обновление контента.

2. **RESTful API**: Серверная часть предоставляет REST API с четким разделением endpoints по функциональности:
   - `/api/search` - поиск
   - `/api/track/<id>` - информация о треке
   - `/api/chart` - чарт
   - `/api/auth/*` - аутентификация
   - `/api/favorites/*` - избранное
   - `/api/history/*` - история

3. **Модульность**: Код разделен на логические модули (app.py для роутинга, database.py для работы с БД, database_api.py для API БД).

4. **Безопасность**: Использование хеширования паролей (Werkzeug), валидация входных данных, защита от SQL-инъекций через параметризованные запросы.

```python
@app.route('/api/track/<track_id>')
def api_track(track_id):
    # Проверка кэша
    cached_track = db.get_cached_track(track_id)
    if cached_track and cached_track.get('stream_url'):
        return jsonify(cached_track)
    
    # Получение трека через API
    track = client.tracks([track_id])[0]
    
    # Получение информации о загрузке
    download_info_list = client.tracks_download_info(track_id, get_direct_links=False)
    
    # Выбор лучшего варианта (mp3, 192+ kbps)
    best_info = None
    for info in download_info_list:
        if info.codec == 'mp3' and info.bitrate_in_kbps >= 192:
            best_info = info
            break
    
    # Получение прямой ссылки
    stream_url = best_info.get_direct_link()
    
    # Формирование данных трека
    track_data = {
        'id': track_id,
        'title': track.title,
        'artists': [{'id': a.id, 'name': a.name} for a in track.artists],
        'stream_url': stream_url,
        'codec': best_info.codec,
        'bitrate': best_info.bitrate_in_kbps
    }
    
    # Кэширование (50 секунд)
    db.cache_track(track_data, expires_in_seconds=50)
    
    # Добавление в историю
    db.add_play_history(track_data, user_id=session.get('user_id') or 0)
    
    return jsonify(track_data)
```

```python
    # Хеширование пароля
```

### Методы работы с данными

```python
def cache_track(self, track_data: Dict, expires_in_seconds: int = 60):
    expires_at = datetime.now() + timedelta(seconds=expires_in_seconds)
    cursor.execute('''
        INSERT OR REPLACE INTO track_cache
        (track_id, title, artists, stream_url, expires_at)
        VALUES (?, ?, ?, ?, ?)
    ''', (track_data['id'], track_data['title'], 
          json.dumps(track_data.get('artists', [])),
          track_data.get('stream_url'), expires_at))

def get_cached_track(self, track_id: str) -> Optional[Dict]:
    cursor.execute('''
        SELECT * FROM track_cache
        WHERE track_id = ? AND (expires_at IS NULL OR expires_at > datetime('now'))
    ''', (track_id,))
    row = cursor.fetchone()
    if row:
        track = dict(row)
        track['artists'] = json.loads(track['artists'])
        return track
    return None
```
