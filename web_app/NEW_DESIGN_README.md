# 🎨 Spectral Music - Новый Дизайн

## 🎉 Что нового?

Ваше музыкальное приложение получило **полную визуальную трансформацию** в стиле **"Dark Kinetic Glassmorphism"**!

---

## ✨ Основные изменения

### 1. 🌈 Визуальный стиль
- **Темная тема** с динамическими градиентами (оранжево-красные тона)
- **Glassmorphism эффекты** — полупрозрачные поверхности с размытием
- **Анимированный фон** — градиент медленно "дышит" и создает глубину
- **Мягкие тени и свечение** — каждый элемент выглядит объемным

### 2. 📝 Типографика
- **Bebas Neue** — крупные заголовки
- **Montserrat** — средние заголовки
- **Manrope** — основной текст
Все шрифты загружаются из Google Fonts автоматически.

### 3. 🎬 Анимации

#### Появление элементов
- Карточки треков появляются последовательно с эффектом "волны"
- Модальные окна "выпрыгивают" с bounce-эффектом
- Страницы плавно переключаются с fade-эффектом

#### Интерактивность
- **Ripple эффект** — при клике на кнопки волна расходится от точки нажатия
- **Hover эффекты** — карточки поднимаются и светятся при наведении
- **Magnetic эффект** — элементы слегка "притягиваются" к курсору
- **Пульсация плеера** — обложка "дышит" во время воспроизведения

### 4. 🎵 Улучшенный плеер
- Glassmorphism с сильным размытием
- Анимированный progress bar с буфером
- Кнопки с градиентным свечением
- Плавные анимации при переключении треков

### 5. 📱 Адаптивность
- Полностью отзывчивый дизайн для всех устройств
- Оптимизированная раскладка для планшетов и мобильных
- Touch-friendly интерфейс

### 6. 🔔 Toast уведомления
- Красивые всплывающие уведомления вместо alert()
- Автоматически исчезают через 3 секунды
- Разные цвета для success/error/info

### 7. ⬆️ Scroll to Top
- Плавающая кнопка для быстрого возврата наверх
- Появляется при прокрутке вниз
- Gradient стиль

## 🐛 Известные особенности

### Производительность
- Для лучшей производительности на слабых устройствах можно отключить анимации в браузере:
  - Chrome: Settings → Accessibility → Prefers reduced motion
  - Firefox: about:config → ui.prefersReducedMotion

### Совместимость
- Лучше всего работает в современных браузерах:
  - Chrome/Edge 90+
  - Firefox 88+
  - Safari 14+

### Glassmorphism
- Эффект размытия (`backdrop-filter`) может не работать в старых браузерах
- В этом случае будет отображаться просто полупрозрачный фон

## 🎨 Кастомизация

### Изменить основной цвет
Откройте `web_app/static/css/style.css` и измените в `:root`:

```css
--color-primary: #your-color;
--gradient-accent: linear-gradient(45deg, #color1, #color2);
```

### Отключить анимации
Добавьте в конец `style.css`:

```css
* {
  animation: none !important;
  transition: none !important;
}
```

### Изменить шрифты
В `web_app/templates/index.html` измените ссылку на Google Fonts:

```html
<link href="https://fonts.googleapis.com/css2?family=YourFont&display=swap" rel="stylesheet">
```
