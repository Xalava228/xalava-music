## ⚡ Быстрый старт (30 секунд)

### 1. Запустите сайт:
```bash
python app.py
```

### 2. Откройте в браузере:
```
http://localhost:8080
```

### 3. Протестируйте адаптивность:
- Нажмите `F12` (откроется DevTools)
- Нажмите `Ctrl+Shift+M` (Device Toolbar)
- Выберите разные устройства из списка
