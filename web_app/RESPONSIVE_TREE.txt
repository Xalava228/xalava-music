🎯 ЧТО ВНУТРИ style-responsive.css:

1. CSS Variables:
   ├── Адаптивные spacing (10-48px)
   ├── Touch target sizes (44px min)
   └── Responsive font sizes

2. Mobile Styles (<768px):
   ├── Вертикальный header
   ├── Horizontal menu scroll
   ├── 1 колонка контента
   ├── 4-рядный плеер
   └── Touch оптимизация

3. Tablet Styles (768-991px):
   ├── Горизонтальный header
   ├── 2 колонки контента
   ├── 2×2 плеер
   └── Улучшенный spacing

4. Desktop Small (992-1199px):
   ├── Вертикальное боковое меню (240px)
   ├── 3 колонки контента
   ├── 3-колоночный плеер
   └── Sticky sidebar

5. Desktop Medium (1200-1599px):
   ├── Широкое боковое меню (280px)
   ├── 6-8 колонок контента
   ├── 4-колоночный плеер (с громкостью)
   └── Оптимальные размеры

6. Desktop Large (≥1600px):
   ├── Максимальные размеры карточек
   ├── 8-12+ колонок
   └── Динамические сетки

7. Специальные медиазапросы:
   ├── @media (max-width: 375px)           ← Очень маленькие
   ├── @media (orientation: landscape)     ← Горизонтальная
   ├── @media (hover: none)                ← Touch устройства
   └── @media (prefers-reduced-motion)     ← Accessibility
