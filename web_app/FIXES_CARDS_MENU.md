# ✅ Исправления: Бургер меню и карточки

## 🍔 Исправление бургер меню

### Проблема:
- Меню было прозрачным и не кликабельным

### Решение:
- ✅ Добавлен `background: var(--color-surface-elevated)` к `.sidebar`
- ✅ Добавлен `backdrop-filter: blur(20px)` для эффекта стекла
- ✅ Добавлен `pointer-events: none` по умолчанию
- ✅ При `.sidebar.active` установлен `pointer-events: auto`

**Файл:** `static/css/burger-menu.css`

---

## 🎴 Переделка карточек

### Что изменилось:

#### 1. **Структура карточки:**
- Изображение теперь **фон карточки** (absolute positioning)
- Название и артист **поверх изображения**
- Кнопки появляются **только при hover** (desktop)

#### 2. **Hover эффекты:**
- Изображение увеличивается и сдвигается вверх (`translateY(-4%)`)
- Gradient overlay "поднимается" для лучшей видимости текста
- Кнопки выезжают снизу с плавной анимацией
- Текст становится ярче

#### 3. **Кнопки:**
- **Избранное** (❤️/🤍) - добавление/удаление из избранного
- **В плейлист** (📋) - добавление в плейлист
- **Удалить** (🗑️) - удаление из плейлиста (если в плейлисте)

#### 4. **Стили кнопок:**
- Темный фон с blur эффектом
- Белая рамка
- При hover: светлее + поднимается вверх
- Минимальный размер 44×44px (touch-friendly)

#### 5. **Мобильные устройства:**
- Кнопки **всегда видны** на мобильных (нет hover)
- Адаптивные размеры
- Touch-friendly

## 📊 Изменённые файлы

### 1. `static/css/burger-menu.css`
```css
.sidebar {
    background: var(--color-surface-elevated);
    backdrop-filter: blur(20px);
    pointer-events: none;
}

.sidebar.active {
    pointer-events: auto;
}
```

### 2. `static/css/style.css`
- Переделан `.track-card` (padding: 0, flex-end)
- `.track-cover-img` теперь absolute (фон)
- `.track-card-title` и `.track-card-artist` поверх изображения
- `.track-actions` скрыты по умолчанию, появляются при hover
- Улучшены стили кнопок

### 3. `static/js/app.js`
- Кнопки всегда создаются (если есть trackId)
- Улучшены title атрибуты для accessibility

## 🧪 Как протестировать

1. **Бургер меню:**
   - Откройте на мобильном/планшете
   - Кликните на `[☰]`
   - Меню должно быть непрозрачным и кликабельным

2. **Карточки:**
   - Найдите треки через поиск
   - Наведите на карточку (desktop)
   - Кнопки должны появиться снизу
   - Изображение должно увеличиться
   - На мобильном кнопки всегда видны
