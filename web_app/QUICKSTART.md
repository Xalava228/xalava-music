# Быстрый старт

## Установка и запуск

1. **Установите зависимости:**
```bash
cd web_app
pip install -r requirements.txt
```

2. **Запустите приложение:**
```bash
python app.py
```

3. **Откройте в браузере:**
```
http://localhost:8080
```

**Примечание:** Если порт 8080 занят, измените его через переменную окружения:
```bash
set PORT=5001
python app.py
```

## Использование

- **Поиск:** Введите запрос в поисковую строку и нажмите "Поиск"
- **Чарт:** Перейдите в раздел "Чарт" в боковом меню
- **Воспроизведение:** Кликните на любой трек для начала воспроизведения

## Получение токена (опционально)

Для полного доступа к функциям:

1. Откройте Яндекс.Музыку в браузере
2. Откройте инструменты разработчика (F12)
3. Перейдите на вкладку "Network"
4. Обновите страницу
5. Найдите запрос к API и скопируйте токен из заголовков
6. Введите токен в поле "Токен" на сайте и нажмите "Войти"

**Без токена:** Доступен только поиск и просмотр информации, воспроизведение ограничено первыми 30 секундами.

## Структура

- `app.py` - Flask сервер
- `templates/index.html` - HTML страница
- `static/css/style.css` - Стили
- `static/js/app.js` - JavaScript логика

## Проблемы?

- **Ошибка доступа к сокету:** Порт может быть занят. Попробуйте другой порт:
  ```bash
  set PORT=5001
  python app.py
  ```
- Проверьте консоль браузера (F12) на наличие ошибок
- Убедитесь, что установлены все зависимости
- Если порт 8080 занят, используйте другой (5001, 3000, и т.д.)
