## 🐛 Если что-то не работает

### Проверьте:
1. **Логи** в Dashboard платформы
2. **Переменные окружения** установлены правильно
3. **Root Directory** = `web_app`
4. **Start Command** правильный
5. **Порт** берется из переменных окружения

### Частые ошибки:

**"Module not found"**
→ Проверьте `requirements.txt`, перезапустите

**"Port already in use"**
→ Используйте переменную `PORT`

**"Application error"**
→ Проверьте логи, убедитесь что `Root Directory` правильный

**"Database locked"**
→ Нормально для SQLite, для продакшена используйте PostgreSQL
