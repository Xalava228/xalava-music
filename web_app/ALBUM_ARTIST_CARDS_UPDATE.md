# ✅ Обновление карточек альбомов и исполнителей

## 🎨 Новый эффект - выезжающая плашка

### Что изменилось:

**ДО:**
- Карточка с изображением, названием и артистом всегда видимыми
- Hover эффект с увеличением изображения

**ПОСЛЕ:**
- Карточка показывает **только изображение**
- При hover **снизу выезжает плашка** с текстом
- Изображение **сдвигается вниз** при hover
- Плашка с темным фоном и blur эффектом

---

## 📊 Структура карточек

### Альбомы:
```html
<div class="album-card-box" style="background-image: url(...)">
    <div class="album-card-footer">
        <div class="album-card-info">
            <strong class="album-card-title">Название альбома</strong>
            <span class="album-card-artist">Исполнитель</span>
        </div>
    </div>
</div>
```

### Исполнители:
```html
<div class="artist-card-box" style="background-image: url(...)">
    <div class="artist-card-footer">
        <div class="artist-card-info">
            <strong class="artist-card-title">Имя исполнителя</strong>
        </div>
    </div>
</div>
```

## 🎬 Анимации

### Обычное состояние:
```
┌──────────────┐
│              │
│   [Image]    │ ← только изображение
│              │
│              │
└──────────────┘
```

### При hover:
```
┌──────────────┐
│              │
│   [Image↓]   │ ← сдвинуто вниз
│              │
│              │
│ ┌──────────┐ │ ← плашка выехала
│ │ Title    │ │
│ │ Artist   │ │
│ └──────────┘ │
└──────────────┘
```

---

## 📐 CSS эффекты

### Изображение:
- `background-position: top center` → `center 100%` при hover
- `background-size: 100% 110%` для плавного сдвига
- Transition: `0.4s ease`

### Плашка:
- `bottom: -50%` → `bottom: 0` при hover
- `background: rgba(0, 0, 0, 0.85)` с blur
- Transition: `0.5s ease`

---

## 📱 Адаптивность

### Desktop (≥600px):
- Плашка скрыта по умолчанию
- Появляется при hover

### Mobile (<600px):
- Плашка **всегда видна** (`bottom: 0`)
- Нет hover эффектов

## 🧪 Как протестировать

1. **Откройте поиск:**
   - Найдите альбомы или исполнителей
   - Карточки должны показывать только изображение

2. **Hover эффект (desktop):**
   - Наведите на карточку
   - Плашка должна выехать снизу
   - Изображение должно сдвинуться вниз

3. **Мобильные:**
   - Плашка должна быть всегда видна
   - Нет hover эффектов
