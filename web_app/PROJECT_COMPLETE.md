## 🚀 Как начать

### Локальная разработка:

1. **Установите зависимости:**
   ```bash
   cd web_app
   pip install -r requirements.txt
   ```

2. **Запустите приложение:**
   ```bash
   python app.py
   ```

3. **Откройте в браузере:**
   ```
   http://localhost:8080
   ```

### Деплой на сервер:
