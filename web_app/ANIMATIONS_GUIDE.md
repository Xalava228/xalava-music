## 🍔 Burger Menu Animations

### Анимация кнопки (3 линии → крестик):

**Характеристики:**
- Easing: `cubic-bezier(.89, 0, .08, 1)`
- Длительность: `0.5s`
- Плавное превращение линий

**Что происходит:**
```
[☰] → [×]
 │     │
Line 1: rotate(135deg) + translateY(7px)
Line 2: width → 0 + rotate(90deg)  
Line 3: rotate(225deg) + translateY(-7px)
```

### Полноэкранное меню:

**Эффект scale:**
- Начальное состояние: `scale(1, 0)` (сжато по Y)
- При открытии: `scale(1, 1)` (полный размер)
- Origin: `left top`
- Easing: `cubic-bezier(.78, 0, .23, 1)`
- Длительность: `0.5s`

**Затемнение контента:**
- Container уменьшается: `scale(0.95)`
- Opacity снижается: `0.3`
- Easing: `cubic-bezier(.49, 0, .52, 1)`
- Длительность: `0.5s`

### Stagger Animation (пункты меню):

**Cascade появление:**
- Каждый пункт появляется с задержкой
- Transform: `translateY(-80px) → translateY(0)`
- Opacity: `0 → 1`
- Easing: `cubic-bezier(.19, .18, .04, 1)`
- Длительность: `0.8s`

**Задержки:**
```
Поиск:        0.15s
Авторизация:  0.2s
Главная:      0.25s
Чарт:         0.3s
Поиск:        0.35s
Подкасты:     0.4s
Избранное:    0.45s
История:      0.5s
Плейлисты:    0.55s
```

**Эффект:**
```
Сначала → ┌────────────┐
          │            │
          │            │
          └────────────┘

Затем →   ┌────────────┐
          │ 🔍 Поиск   │ ← появляется
          │ 🔐 Войти   │ ← появляется
          │ • Главная  │ ← появляется
          │ • Чарт     │ ← появляется
          └────────────┘
```

## 🎴 Card Hover Effects

### Gradient Overlay:

**Технология:**
- Двухслойный gradient overlay
- Transform: `translateY(0) → translateY(-50%)`
- Создаёт эффект "поднятия" контента

**Градиент:**
```css
linear-gradient(
    to bottom, 
    rgba(0, 0, 0, 0) 0%,      ← Прозрачный
    rgba(0, 0, 0, 0.3) 50%,   ← Полупрозрачный
    rgba(0, 0, 0, 0.79) 100%  ← Темный
)
```

### Image Zoom:

**Эффект:**
- Image scale: `1 → 1.1`
- Image translateY: `0 → -4%`
- Easing: `cubic-bezier(0.19, 1, 0.32, 1)`
- Длительность: `1.35s`

**Создаёт:**
- Плавный zoom с небольшим сдвигом вверх
- Профессиональный "cinematic" эффект

### Content Reveal:

**На десктопах (hover: hover):**
- Начальное состояние:
  - Текст: `opacity: 0.7`, `translateY(4px)`
  - Кнопки: скрыты

- При hover:
  - Текст: `opacity: 1`, `translateY(0)`
  - Кнопки: появляются
  - Delay: `0.1s`

**На мобильных:**
- Контент всегда видимый
- Анимации отключены

## 🎨 Применённые easing функции

| Easing | Применение | Характер |
|--------|-----------|----------|
| `cubic-bezier(.89, 0, .08, 1)` | Burger lines | Медленный старт, быстрый конец |
| `cubic-bezier(.78, 0, .23, 1)` | Menu scale | Плавное масштабирование |
| `cubic-bezier(.49, 0, .52, 1)` | Content fade | Сбалансированное затемнение |
| `cubic-bezier(.19, .18, .04, 1)` | Menu items | Bouncy появление |
| `cubic-bezier(0.19, 1, 0.32, 1)` | Card hover | Smooth zoom |

## 💡 Особенности

### Mobile First:
- На мобильных контент всегда видимый
- Hover эффекты только на `(hover: hover) and (min-width: 600px)`
- Touch-friendly без лишних анимаций

### Performance:
- Используются только transform и opacity (GPU-accelerated)
- Нет анимаций width/height (медленные)
- Оптимизированные easing функции

### Accessibility:
- Анимации отключаются при `prefers-reduced-motion`
- Плавные переходы не создают дискомфорт
- Логичная последовательность

## 🔧 Кастомизация

### Изменить скорость burger menu:

```css
/* burger-menu.css */
.sidebar {
    transition: transform cubic-bezier(.78, 0, .23, 1) 0.5s;
    /* Измените 0.5s */
}
```

### Изменить delay для пунктов меню:

```css
/* burger-menu.css */
.sidebar.active .nav-item:nth-child(1) { 
    transition-delay: 0.25s; /* Измените */
}
```

### Изменить скорость image zoom:

```css
/* style.css */
.track-cover img {
    transition: transform 1.35s cubic-bezier(0.19, 1, 0.32, 1);
    /* Измените 1.35s */
}
```
