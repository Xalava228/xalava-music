# Yandex Music Web Player

Веб-приложение для прослушивания музыки через Yandex Music API. Создано для курсовой работы на основе библиотеки [yandex-music-api](https://github.com/MarshalX/yandex-music-api).

## Возможности

- 🎵 Поиск треков, альбомов и исполнителей
- 📊 Просмотр чарта популярной музыки
- ▶️ Воспроизведение треков в браузере
- 🎨 Современный и красивый интерфейс
- 📱 Адаптивный дизайн для мобильных устройств
- 🔐 Поддержка авторизации через токен (опционально)

## Требования

- Python 3.7 или выше
- pip (менеджер пакетов Python)

## Установка

1. Перейдите в директорию веб-приложения:
```bash
cd web_app
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

Или если используете виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

1. Запустите Flask приложение:
```bash
python app.py
```

2. Откройте браузер и перейдите по адресу:
```
http://localhost:8080
```

**Примечание:** Если порт 8080 занят, вы можете изменить его, установив переменную окружения `PORT`:
```bash
set PORT=5001
python app.py
```

## Получение токена (опционально)

Для полного доступа к функциям (воспроизведение полных треков, доступ к личным плейлистам) вам нужен токен Яндекс.Музыки.

### Как получить токен:

1. Откройте браузер и войдите в свой аккаунт Яндекс.Музыки
2. Откройте инструменты разработчика (F12)
3. Перейдите на вкладку "Network" (Сеть)
4. Обновите страницу (F5)
5. Найдите запрос к API (например, к `music.yandex.ru/api/...`)
6. В заголовках запроса найдите `Authorization` или `Cookie`
7. Скопируйте токен (обычно это строка из 39 символов)

**Альтернативный способ:**
- Используйте расширения браузера для получения токена
- Или следуйте инструкциям в [документации библиотеки](https://yandex-music.readthedocs.io/en/latest/token.html)

### Использование токена:

После запуска приложения введите токен в поле "Токен" в шапке сайта и нажмите "Войти".

**Примечание:** Без токена вы можете искать треки и просматривать информацию, но воспроизведение может быть ограничено (только первые 30 секунд).

## API Endpoints

Приложение предоставляет следующие API endpoints:

- `GET /` - Главная страница
- `POST /api/init` - Инициализация клиента с токеном
- `GET /api/search?q=<query>` - Поиск треков, альбомов, исполнителей
- `GET /api/track/<track_id>` - Получение информации о треке и ссылки для воспроизведения
- `GET /api/album/<album_id>` - Получение информации об альбоме и его треках
- `GET /api/chart` - Получение чарта популярной музыки

## Использование

1. **Поиск музыки:**
   - Введите запрос в поисковую строку
   - Нажмите "Поиск" или Enter
   - Выберите трек из результатов для воспроизведения

2. **Просмотр чарта:**
   - Перейдите в раздел "Чарт" в боковом меню
   - Выберите трек для воспроизведения

3. **Управление плеером:**
   - Используйте кнопки управления в нижней панели
   - Регулируйте громкость и прогресс воспроизведения

## Технологии

- **Backend:** Flask (Python)
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **API:** yandex-music-api
- **Стили:** Современный CSS с градиентами и анимациями

## Особенности реализации

- Использование неофициального API Яндекс.Музыки
- Прямое воспроизведение треков через HTML5 Audio API
- Адаптивный дизайн для различных устройств
- Обработка ошибок и состояний загрузки
- Кэширование плейлистов для навигации между треками

## Ограничения

- Без токена доступны только первые 30 секунд треков
- Некоторые функции требуют авторизации
- Прямые ссылки на треки действительны ограниченное время (около 1 минуты)
