### 2. ✅ Улучшена адаптация плеера

**Проблема:** Плеер занимал слишком много места, перекрывал контент

**Решение для мобильных (< 768px):**

**ДО:**
```
┌─────────────────────┐
│ Progress            │ ← 4 ряда
│ Info                │
│ Controls            │
│ Volume              │
└─────────────────────┘
```

**ПОСЛЕ:**
```
┌─────────────────────┐
│ [🎵] Track | ⏯️ ⏭️  │ ← 2 ряда (компактно!)
│ ═══════════════     │
└─────────────────────┘
```

**Изменения:**
- ✅ Объединены Info и Controls в одну строку
- ✅ Скрыты shuffle и repeat на мобильных
- ✅ Скрыта громкость на мобильных
- ✅ Уменьшены размеры элементов (44px вместо 52px)
- ✅ Уменьшены отступы
- ✅ Прогресс-бар стал тоньше (4px вместо 6px)

**Решение для планшетов (768-991px):**

- ✅ Более компактная сетка (3 колонки)
- ✅ Info слева (max-width: 200px)
- ✅ Controls по центру
- ✅ Progress на всю ширину снизу
- ✅ Громкость скрыта для компактности
- ✅ Shuffle и repeat показываются

## 🔍 Отладка

Если меню всё ещё не закрывается:

1. Откройте консоль браузера (F12)
2. Попробуйте закрыть меню
3. Проверьте сообщения в консоли:
   - `"Close button clicked"` - кнопка работает
   - `"closeSidebar called"` - функция вызывается
   - `"Sidebar closed successfully"` - успешное закрытие

Если видите ошибки - сообщите, что именно выводится в консоль.
