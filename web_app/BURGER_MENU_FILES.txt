🎨 ВИЗУАЛЬНЫЕ ЭЛЕМЕНТЫ:
────────────────────────────────────────────────────────────────

1. Кнопка бургер [☰]
   - 3 линии → превращаются в крестик [×]
   - Цвет: полупрозрачный белый
   - Hover: оранжевая подсветка

2. Sidebar (выдвижная панель)
   - Ширина: 320px (или 85vw на маленьких)
   - Фон: glassmorphism с blur
   - Анимация: выезжает слева

3. Overlay (затемнение)
   - Фон: rgba(0, 0, 0, 0.6) + blur(4px)
   - Клик закрывает меню

4. Шапка sidebar
   - "Навигация" + кнопка закрытия [×]
   - Только на mobile/tablet
