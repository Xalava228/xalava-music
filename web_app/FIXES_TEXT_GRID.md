# ✅ Исправления: Текст на карточках и количество колонок

## 🎨 Проблема с белым текстом на белом фоне

### Проблема:
- Белый текст на белом фоне изображения сливался
- Текст был виден всегда, даже когда не читаем

### Решение:
- ✅ Текст **скрыт по умолчанию** (`opacity: 0`)
- ✅ Текст **появляется при hover** когда карточка затемнена
- ✅ Плавная анимация появления (`opacity` + `translateY`)
- ✅ На мобильных текст **всегда виден** (нет hover)

**Файл:** `static/css/style.css`

---

## 📐 Изменение количества колонок

### Было:
- `repeat(auto-fill, minmax(...))` - автоматически подстраивалось
- Могло быть 7+ треков в строке

### Стало:
- `repeat(6, 1fr)` - **ровно 6 колонок** на всех экранах ≥768px
- На мобильных остается `auto-fill` для адаптивности

**Файлы:**
- `static/css/style.css`
- `static/css/style-responsive.css`

---

## 📊 Детали изменений

### 1. Скрытие текста:

```css
/* По умолчанию - скрыт */
.track-card-title,
.track-card-artist {
    opacity: 0;
    transform: translateY(10px);
    transition: opacity 0.6s, transform 0.6s;
}

/* При hover - появляется */
.track-card:hover .track-card-title,
.track-card:hover .track-card-artist {
    opacity: 1;
    transform: translateY(0);
}

/* На мобильных - всегда виден */
@media (max-width: 599px) {
    .track-card-title,
    .track-card-artist {
        opacity: 1;
        transform: translateY(0);
    }
}
```

### 2. Grid колонки:

```css
/* Мобильные (<768px) - адаптивно */
.tracks-grid {
    grid-template-columns: repeat(auto-fill, minmax(140px, 1fr));
}

/* Планшеты и десктопы (≥768px) - 6 колонок */
@media (min-width: 768px) {
    .tracks-grid {
        grid-template-columns: repeat(6, 1fr);
    }
}
```

## 🧪 Как протестировать

1. **Текст:**
   - Найдите треки через поиск
   - Наведите на карточку (desktop)
   - Текст должен появиться плавно
   - На мобильном текст всегда виден

2. **Колонки:**
   - Откройте на десктопе/планшете
   - Должно быть ровно 6 треков в строке
   - На мобильном - адаптивно
