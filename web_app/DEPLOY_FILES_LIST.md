### Procfile:
```

```
Используется на: Heroku, Render (опционально)

### runtime.txt:
```

```
Используется на: Heroku, некоторые другие

### Dockerfile:
Используется на: Fly.io, Railway (опционально)

### render.yaml:
Используется на: Render.com (автоматическая настройка)

### .gitignore:
Исключает из Git:
- Видео файлы (*.mp4)
- База данных (*.db)
- Временные файлы
- Секреты (.env)
