## 🔧 Технические детали

### CSS Подход:
- **Mobile First** - базовые стили для <768px, затем медиазапросы
- **Progressive Enhancement** - добавление функций для больших экранов
- **CSS Grid & Flexbox** - современные layout техники
- **CSS Variables** - адаптивные spacing и sizing

### Breakpoints:
```css
/* Base: Mobile (<768px) */
/* Стили по умолчанию */

@media (min-width: 768px) { /* Tablet */ }
@media (min-width: 992px) { /* Desktop Small */ }
@media (min-width: 1200px) { /* Desktop Medium */ }
@media (min-width: 1600px) { /* Desktop Large */ }
```

### Специальные медиазапросы:
```css
@media (max-width: 375px) { /* Very small phones */ }
@media (max-width: 768px) and (orientation: landscape) { /* Landscape mobile */ }
@media (hover: none) and (pointer: coarse) { /* Touch devices */ }
@media (prefers-reduced-motion: reduce) { /* Accessibility */ }
```

## 📞 Поддержка

### Если что-то не работает:
1. Проверьте, что файл `style-responsive.css` подключен в HTML
2. Очистите кеш браузера (Ctrl+Shift+R или Cmd+Shift+R)
3. Проверьте консоль браузера на ошибки (F12)
4. Убедитесь, что оба CSS файла загружаются (Network tab в DevTools)

### Проверка подключения:
1. Откройте DevTools (F12)
2. Перейдите на вкладку **Network**
3. Обновите страницу (F5)
4. Найдите `style-responsive.css` в списке (должен быть статус 200)
